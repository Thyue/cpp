[C++]資料型態

由於資料的型態各不相同，在儲存時所需要的容量不一，不同的 資料必須要配給不同的空間大小來儲存

下列出現「word」一詞先想像為一個單位用量，在32位元機器上通常一個word是4個位元組。

下列只顯示目前有上課進度有教到的，未來有新增在做更新。

    常見的資料型態如下：
  • 整數：目前只有教到int(未來有教再新增)，int表示一個word。
  • 浮點數：用來表示小數值，可以區分為float、double，float的長度為一個word，double的長度為二個word。
  • 布林：用來表示布林數，在程式中佔一個位元組，可以使用true或false 字面常量(Literal constant)來表示，實際上的數值則分別對應於1與0

以上的資料型態在記憶體中所佔有的大小依平台系統而有所差異，最好的作法是使用sizeof()運算子，它可以告訴您確實的記憶體大小。

#include <iostream>
using namespace std;
int main() { 
  
  cout << "int\t" << sizeof(int) << endl; 
  cout << "float\t" << sizeof(float) << endl; 
  cout << "double\t" << sizeof(double) << endl; 
  
  cout << "sizeof(bool)\t" << sizeof(bool) << endl;
  cout << "sizeof(true)\t" << sizeof(true) << endl;
  cout << "sizeof(false)\t" << sizeof(false) << endl;
  cout << "true\t" << static_cast<int>(true) << endl;
  cout << "false\t" << static_cast<int>(false);
  
}
輸出結果：
int 4
float 4
double  8
sizeof(bool)  1
sizeof(true)  1
sizeof(false) 1
true  1
false 0

[C++]變數

變數（Variable）提供一個有名稱的記憶體儲存空間，一個變數關係至一個資料型態、一個變數本身的值與一個變數的位址值。

    在C++中要使用變數，必須先宣告變數名稱與資料型態
  • 首先定義資料型態，我們會使用到「宣告工具」，如下：int、double。
  • 接下來命名變數名稱，命名時盡量與此數值用途有關有意義的名詞。
  • 大小寫之間是有差異的，視為兩個不同的變數名稱，如：Num不等於num。
  • 若有兩個字以上可使用蛇式：current_temp_c，或者駝峰式CurrentTempC命名
  • 不可以使用數字作為開頭。不可以使用一些特殊字元，像是*&^%之類的字元。變數名稱不可以與C++內定的關鍵字(Keyword)同名
  • 可以先宣告空間(第4行)，在賦值(第5行)。或者宣告完當下賦值(第7行)
#include <iostream>
int main() { 

  int intNum;      // 宣告一個整數變數
  intNum = 3;

  double dblNum = 3.1445926;   // 宣告一個倍精度浮點數變數

}

[C++]算術運算

在C++中提供與算術相關的加(+)、減(-)、乘(*)、除(/)的運算子，另外還有一個也很常用的餘除運算子(%)或稱模數(Modulus)運算子，這類以數學運算為主的運算子，稱之為「算術運算子」(Arithmetic operator)。

◎ 四則運算
#include <iostream>
using namespace std;
int main() { 
  
  printf("%d\n", n + 10);  // 110
  printf("%d\n", n - 10);  // 90
  printf("%d\n", n * 10);  // 1000
  printf("%d\n", n / 10);  // 10
  printf("%d\n", n % 10);  // 0
  
}
◎ 運算規則
    其計算方法跟在現實中數學算是一樣：
  • 由左至右進行運算。
  • 先乘除後加減。
  • 必要時加上括號表示運算的先後順序。
#include <iostream>
using namespace std;
int main() { 
  
  cout << (1+5*3)/4 << endl;  // 4
  
}
◎ 餘除運算子(%)

用來計算除法後的餘數，通常來作循環計數之用。

#include <iostream>

[C++]關係運算&條件運算

◎ 關係運算子

C++中也提供了「關係運算子」(Relational operator)或「比較運算子」(Comparison operator)。

  • 大於( > )
  • 不小於( >= )
  • 小於( < )
  • 不大於( <= )
  • 等於 ( == )
  • 不等於( != )
  • 所有非零的數值在作為條件式時都被視為true。
  • 這裡有個地方會有人常常犯錯，「x == y」是有兩個變數x與y要比較是否相等；「x = y」是將y的值指定給x，寫的時候務必要檢查。
#include <iostream>
using namespace std;
int main() { 

  cout << "10 > 5\t\t" << (10 > 5) << endl;
  cout << "10 >= 5\t\t" << (10 >= 5) << endl;
  cout << "10<< 5\t\t" << (10 < 5) << endl;
  cout << "10<<= 5\t\t" << (10 <= 5) << endl;
  cout << "10 == 5\t\t" << (10 == 5) << endl;
  cout << "10 != 5\t\t" << (10 != 5) << endl;

}
輸出結果：
10 > 5          1
10 >= 5        1
10 < 5          0
10 <= 5        0
10 == 5        0
10 != 5         1
◎ 條件運算子

「條件運算子」（Conditional operator）的傳回值依條件式的結果而定，如果條件式的結果為true，則傳回冒號前的值，若為false，則傳回冒號後的值。

#include <iostream>
using namespace std;
int main() { 

  int score = 0; 

  cout << "輸入學生分數："; 
  cin >> score; 
  cout << "該生是否及格？" << (score >= 60 ? 'Y' : 'N') << endl; 

}
輸出結果：
輸入學生分數：60
該生是否及格？Y

[C++]邏輯運算&位元運算

◎ 邏輯運算子

在C++中也提供「邏輯運算子」(Logical operator)。

  • 且 ( && )
  • 或 ( || )
  • 反相 ( ! )
    下面三段程式分別會輸出1、0與1，也就是分別表示true、false與true三種狀況。
  • 「&&」運算中，如果左邊的式子已經被評斷為false，則可立即判斷整個式子為false，因而右邊的式子就不會再評斷。
  • 「||」運算中如果左邊的式子已經被評斷為true，則可以判斷整個式子為true，因而右邊的式子就不會再評斷。
#include <iostream>
using namespace std;
int main() { 

  int num = 75;
  cout <<  (num > 70 && num < 80) << endl;
  cout << (num > 80 || num < 75) << endl;
  cout << !(num > 80 || num < 75) << endl;

}
輸出結果：
1
0
1
◎ 位元運算子

在C++中提供「位元運算子」(Bitwise operator)，分別是AND(&)、OR(|)、NOT(!)、XOR(^)與補數(~)。

  • AND ( & )
  • OR ( | )
  • NOT ( ! )
  • XOR ( ^ )
  • 補數 ( ~ )
#include <iostream>
using namespace std;
int main() { 

  cout << "AND運算：" << endl; 
  cout << "0 AND 0\t\t" << (0 & 0) << endl; 
  cout << "0 AND 1\t\t" << (0 & 1) << endl; 
  cout << "1 AND 0\t\t" << (1 & 0) << endl; 
  cout << "1 AND 1\t\t" << (1 & 1) << endl; 

  cout << "OR運算：" << endl; 
  cout << "0 OR 0\t\t" << (0 | 0) << endl; 
  cout << "0 OR 1\t\t" << (0 | 1) << endl; 
  cout << "1 OR 0\t\t" << (1 | 0) << endl; 
  cout << "1 OR 1\t\t" << (1 | 1) << endl; 

  cout << "XOR運算：" << endl; 
  cout << "0 XOR 0\t\t" << (0 ^ 0) << endl; 
  cout << "0 XOR 1\t\t" << (0 ^ 1) << endl; 
  cout << "1 XOR 0\t\t" << (1 ^ 0) << endl; 
  cout << "1 XOR 1\t\t" << (1 ^ 1) << endl; 

  cout << "NOT運算：" << endl; 
  cout << "NOT 0\t\t" << (!0) << endl; 
  cout << "NOT 1\t\t" << (!1) << endl; 

}
輸出結果：
AND運算：

0 AND 1         0
1 AND 0         0
1 AND 1         1
OR運算：
0 OR 0          0
0 OR 1          1

1 OR 1          1
XOR運算：
0 XOR 0         0
0 XOR 1         1
1 XOR 0         1
1 XOR 1         0
NOT運算：
NOT 0           1
NOT 1           0
    總結一下知識
  • 邏輯運算子與位元運算子也是很常被混淆，「&&」為邏輯運算，而「&」為位元運算，「||」為邏 輯運算，而「|」為位元運算。
  • 位元運算是逐位元運算的，例如：10010001與01000001作AND運算，是一個一個位元對應運算，答案就是00000001。
  • 補數運算是將所有的位元0變1，1變0，例如00000001經補數運算就會變為11111110。

位元運算對初學者來說的確較不常用，但如果用的洽當的話，可以增進不少程式效率。

    下面這個程式原理：
  • 奇數的數值若以二進位來表示，其最右邊的位元必為1，而偶數最右邊的位元必為0。
  • 使用1來與輸入的值作AND運算，由於 1除了最右邊的位元為1之外，其它位元都會是0，與輸入數值AND運算的結果，只會留下最右邊位元為0或為的結果，其它部份都被0 AND運算遮掉了，這就是所謂「位元遮罩」。 
#include <iostream>
using namespace std;
int main() { 

  int input = 0; 

  cout << "輸入正整數："; 
  cin >> input; 
  cout << "輸入為奇數？" << (input&1 ? 'Y' : 'N') << endl; 

}
輸出結果：
輸入正整數：5
輸入為奇數？Y

[C++]if條件判斷

    C++提供了if 條件判斷陳述
  • 就是當條件式成立時（true），則執行陳述句一，要不然就執行陳述句二。
  • 如果條件式不成立時並不想作任何事，則else可以 省略。
#include <iostream>
using namespace std;
int main() { 

  // 第一種縮排方法
  if ( /*條件式*/ ) cout << "陳述句一";
  else  cout << "陳述句二";

  // 第二種縮排方法
  if ( /*條件式*/ ) 
    cout << "陳述句一";
  else
    cout << "陳述句二";

}

在 if 後如果有兩個以上陳述句，稱之為「複合陳述句」(compound statement)，此時您必須使用{ }將複合陳述句包括起來。

#include <iostream>
using namespace std;
int main() { 

  if ( /*條件式*/ ) {
    cout << "陳述句一";
    cout << "陳述句二";
  }
  else {
    cout << "陳述句三";
    cout << "陳述句四";
  }

}

當然人生一個問題總是會面臨不同的選擇，如果條件式一不滿足，就會執行條件式二的測試，如果滿足就執行條件式二，以此類推，若都不是則執行else中的陳述 

#include <iostream>
using namespace std;
int main() { 

  if ( /*條件式一*/ ){
    cout << "陳述句一";
    cout << "陳述句二";
  }
  else if ( /*條件式二*/ ){
    cout << "陳述句三";
    cout << "陳述句四";
  }
  else if ( /*條件式三*/ ){
    cout << "陳述句五";
    cout << "陳述句六";
  }
  else {
    cout << "陳述句五";
    cout << "陳述句六";
  }

}

[C++]switch條件判斷

    switch是C++提供的條件判斷陳述式，它只能比較數值或字元，不過別以為這樣它就比if 來得沒用，使用適當的話，它可比if判斷式來得有效率
  • 首先看看switch的括號，當中置放您要取出數值的變數。
  • 取出數值之後，程式會開始與case中所設定的數字或字元作比對，如果符合就執行以下的陳述句。
  • 直到遇到break後離開switch區塊。
  • 如果沒有符合的數值或字元，則會執行default後的陳述句，default不一定需要，您可以省去這個部份。
#include <iostream>
using namespace std;
int main() { 

  switch ( /*變數名稱或運算式*/ ) {
    case /*符合數字或字元*/ :
      cout << "陳述句一";
      break;
    case /*符合數字或字元*/ :
      cout << "陳述句二";
      break;
    default:
      cout << "陳述句三";
  }

}

#include <iostream>

其數學原理使用積分方法計算，如下：

  • 上圖是一個0~1的數線，其平均值為1/2。
  • 在數線上每一個可能的數字距離1/2的平方加起來，就是它的變異值。
  • 因為中間的每個數字發生的機率都一樣，所以在數學上，其變異值為： 。
  • 一共有12個數字數字加起來。
  • ：變異值，並且對x進行微分(dx)。
  • 換算後為等式右邊之結果。
  • 1代入：1/3*(1-1/2)^3=1/24。
  • 0代入：1/3*(0-1/2)^3=-1/24。
  • 結果：1/24-(-1/24)=1/12。
  • 我們將變異值為1的這個結果稱之為常態分佈，特點就是取12次平均分佈。
