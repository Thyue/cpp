[C++]結構：struct

struct 是 C 中用來組織資料的關鍵字，當使用 struct 時，考慮的是一組資料擁有的相關性。

    以「學生」舉個例子：
  • 學生會有姓名、年級、學號、住址、手機號碼等。
  • 若操作時經常都要傳遞同樣一組資料，可以使用 struct 來定義一個 Student型態。
  • 這個型態中包括了姓名(字串)、年級(float)、學號(字串)等資訊。
  • 接著可以使用 Student 來宣告新的資料，進行資料指定或取出等。
#include <iostream>
using namespace std;
struct student {
  int id;         // 系統編號
  char name[10];  // 名字
  int grade;      // 年級
  int student_ID; // 學號
}
int main() { 

  student sam, john;
  sam.id = 0;              // 直接賦予
  scanf("%d", &john.id);   // 從螢幕讀取
  printf("Sam的ID:%d\n", sam.id);
  printf("John的ID:%d\n", john.id);

}
輸出結果：
1
Sam的ID:0
John的ID:1

以下是快速建立10位1年級的學生。

struct student {
  int id;         // 系統編號
  char name[10];  // 名字
  int grade;      // 年級
  int student_ID; // 學號
}
int main() { 

  student sss[10];
    for (int i = 0; i < 10; i++) {
      sss[i].id = i;
      sss[i].grade = 1;
      sss[i].student_ID = 2108410000 + i;

    }

[C++]動態記憶體

◎ 動態記憶體

假如有一間銀行，他們有一套系統，一開始程式在設計時需求只有100人。後來需要1000人的規模，這時候就跟程式公司要求更改至1000人的配置，當然就是一筆費用，經過多次的更改，想必銀行花費不少。

以下是一個動態記憶體設計的例子：

  • 我們先看建立一個帳戶需要多少記憶體。
  • 可以看出是20位元組。
#include <iostream>
using namespace std;
struct customer {
    int id;        // 系統編號
    char name[11]; // 名字
    float amount;  // 餘額
};
int main() { 

  customer *account; // 帳戶的位置
  printf("%lu\n", sizeof(customer));

}
輸出結果：
20
    下面就是一個良好的程式設計
  • 需要多少帳戶，要多少記憶體。
  • 最後釋放記憶體，避免佔用導致當機。
#include <iostream>
using namespace std;
struct customer {
  int id;        // 系統編號
  char name[11]; // 名字
  float amount;  // 餘額
};
int main() { 
  
  cout << "請輸入需要幾個帳戶：" << endl;
  int num_people;              // n個帳戶
  scanf("%d", &num_people);
  customer *account;           // 帳戶的位置
  account = (customer *)malloc(num_people * sizeof(customer)); // 跟記憶體要n個帳戶所需的位元組，存到帳戶的位置

  //賦予帳號基本參數
  for (int i = 0; i << num_people; i++) {
      account[i].id = i;
      account[i].amount = 0.0;
      printf("帳戶%4d，餘額：%4.2f元\n", account[i].id, account[i].amount);
  }

  free(account); // 歸還記憶體

}
